# Renewable Energy Management Portfolio

A modern, dark-themed portfolio website showcasing Renewable Energy Management (REM) coursework for Ayush Kaute.

## 🌱 Live Demo

This website is designed to be hosted on GitHub Pages.

## 📁 Project Structure

```
kaute-rem-portfolio/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Stylesheet with dark mode design
├── js/
│   └── main.js         # JavaScript for interactivity
├── data/               # Folder for assignments, case studies, etc.
│   ├── assignments/
│   ├── casestudies/
│   ├── posters/
│   ├── presentations/
│   └── research/
└── images/             # Folder for images (if needed)
```

## 🚀 How to Deploy on GitHub Pages

1. **Create a GitHub Repository**
   - Go to GitHub and create a new repository (e.g., `rem-portfolio`)

2. **Upload Files**
   - Upload all files from this folder to your repository
   - Make sure to maintain the folder structure

3. **Enable GitHub Pages**
   - Go to repository Settings → Pages
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click Save

4. **Access Your Site**
   - Your site will be live at `https://yourusername.github.io/rem-portfolio`

## 📄 Adding Your Documents

Place your PDFs and documents in the appropriate `data/` subfolders:

- **Assignments**: `data/assignments/solar-pv-design.pdf`
- **Case Studies**: `data/casestudies/denmark-wind.pdf`
- **Posters**: `data/posters/green-hydrogen.pdf`
- **Presentations**: `data/presentations/smart-grid.pptx`
- **Research**: `data/research/perovskite-cells.pdf`

The download links in the Work Showcase section will automatically work once files are added.

## 🎨 Features

- **Hero Section**: Animated energy orb with floating icons
- **About Me**: Personal introduction and mission/vision/values
- **What I Study**: Interactive cards for REM topics
- **Work Showcase**: Expandable dropdown categories
- **Quiz Section**: 10-question interactive quiz
- **Contact Form**: Functional contact section
- **Responsive Design**: Works on all devices
